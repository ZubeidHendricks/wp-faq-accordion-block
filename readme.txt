Build collapsible FAQ accordions with a shortcode — and automatically output FAQ schema for Google rich results.

== Description ==

Create clean, accessible FAQ accordions with a simple shortcode, and let the
plugin emit the FAQPage structured data Google uses to show rich results in
search. Built on native <details> elements — fast, accessible, no jQuery.

    [faq]
      [faq_item q="Do you offer refunds?"]Yes, within 30 days.[/faq_item]
      [faq_item q="Where are you based?"]Cape Town, South Africa.[/faq_item]
    [faq]

**Free features**

* Unlimited accordions and questions.
* Automatic FAQPage schema (rich results).
* Custom accent colour, open-first option.
* Accessible, no jQuery.

**Pro features**

* Visual builder (no shortcodes).
* Search/filter within large FAQ sets.
* Priority support.

== Installation ==

1. Install via Plugins → Add New, or upload the zip.
2. Activate.
3. Add the [faq] / [faq_item] shortcodes to any content.

== Frequently Asked Questions ==

= Does it really get me rich results? =

It outputs valid FAQPage schema. Whether Google shows rich results is always at
Google's discretion, but the structured data is correct and validates.
